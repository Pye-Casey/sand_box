Pointers and Functions
    Passing a pointer as a parameter to a function enables the callee to have the same access to the value as the caller. Any changes made to the dereferenced pointer are reflected in the callerâ€™s value.
Example:
    void setTwenty(int * pNumber)
    {
        *pNumber = 20;
    }

    int main()
    {
        int num = 10;
        int * pNum = &num;
        cout << "Initial Num value: " << num << endl; // outputs 10
        setTwenty(pNum);

        cout << "Num value: " << num << endl; // outputs 20
        return 0;
    }
